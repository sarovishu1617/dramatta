# Dramatta - OTT Platform Clone

**Dramatta** is a web-based OTT platform clone inspired by Aha, built by Saravana Kumar and team.

## 💻 Tech Stack

### Backend:
- Python
- Flask
- SQLite (or JSON for demo)

### Frontend:
- HTML5
- CSS3
- JavaScript (react js)

## 📁 Folder Structure

```
dramatta/
├── backend/
│   └── app/
│       ├── routes/
│       ├── models/
│       ├── templates/
│       └── static/
├── frontend/
│   ├── html files
│   ├── css/
│   ├── js/
│   └── assets/
```
